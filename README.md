# 🚀 AI Resume Builder (5-Day Project Challenge)

Welcome to the **AI Resume Builder**, a full-stack application designed to help users create professional, ATS-friendly resumes quickly and effortlessly using Artificial Intelligence. This project is being built over a span of **5 Days**!

## ✨ Features
- 📝 **Easy-to-use Form**: Interactive forms to input Personal Info, Education, Experience, Skills, and Projects.
- 🎨 **Modern Templates**: Choose from a variety of beautifully designed, premium templates (Classic, Modern, Minimal, etc.).
- 🤖 **AI Integration**: Automatically analyze and enhance your resume using AI.
- 🔒 **Secure Authentication**: Secure user login and registration using JWT.
- 📱 **Fully Responsive**: Beautiful UI/UX that works seamlessly on desktop and mobile.

## 🛠️ Tech Stack
- **Frontend**: React (Vite), Tailwind CSS (or custom CSS), Redux Toolkit
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **AI Integration**: Google Gemini API (or similar)

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/1manish123/ai-resume-builder-.git
cd ai-resume-builder-
```

### 2. Install Dependencies
**For Frontend:**
```bash
cd resume-builder/client
npm install
```

**For Backend:**
```bash
cd resume-builder/server
npm install
```

### 3. Setup Environment Variables
Create a `.env` file in both `client` and `server` folders and add the necessary API keys (MongoDB URI, JWT Secret, AI API Keys, etc.).

### 4. Run the Project
Start both the client and server:
```bash
# In client directory
npm run dev

# In server directory
npm start
```
